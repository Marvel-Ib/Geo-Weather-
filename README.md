# Geo Weather
Node.js app made with Express and Bootstrap  that uses  [Darksky](https://darksky.net/) and  
[Mapbox](https://www.mapbox.com/) 

# Usage
## Installation
Install the dependencies

``$ npm install``

## Serve
To serve in the browser
``$ npm start``
